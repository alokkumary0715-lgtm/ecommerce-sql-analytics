# E-commerce SQL Analytics Project

## 📌 Objective
Analyze customer behavior, revenue trends, and product performance using SQL.

## 📊 Dataset
Relational schema with:
- Customers
- Orders
- Order Items
- Products
- Payments

## ⚙️ Tools Used
- PostgreSQL
- SQL (Advanced Queries)
- pgAdmin

## 🔑 Key Analysis
- Revenue trends & monthly growth
- Top customers & product categories

## 📈 Key Insights
- Top 20% customers contribute majority of revenue
- Revenue shows strong monthly growth trend
- Repeat customers drive higher profitability
- Electronics category dominates revenue

## 🚀 Skills Demonstrated
- Data Modeling
- SQL (JOINs, CTEs, Window Functions)
- Business Analysis
- Performance Optimization
